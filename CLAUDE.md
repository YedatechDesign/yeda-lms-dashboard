# LMS Dashboard Template — הוראות לקלוד

## לפני יצירת דשבורד חדש — שאל תמיד:

### שאלה 1: סוג הלקוח
**האם הדשבורד מיועד לארגון/חברה או למכללה/מוסד לימוד?**

#### אם מוסד לימודי / מכללה:
- **שמור** את המינוח הנוכחי: קורסים, סטודנטים/לומדים, מבחנים, לומדות
- **שמור** את צבעי המותג הנוכחיים (כחול ראשי)

#### אם ארגון / חברה:
- **שנה מינוח:**
  - קורסים ← הכשרות / הדרכות
  - סטודנטים ← עובדים / משתמשים
  - מבחנים ← תרגילים / משימות
- **שנה צבע ראשי:** החלף כחול `#0A59EB` בכתום `#F08700` כצבע הדומיננטי

---

## צבעי מותג
| שם | HEX | שימוש |
|---|---|---|
| Navy | `#000F61` | כותרות, header |
| Blue | `#0A59EB` | ראשי (מוסדות לימוד) |
| Light Blue | `#079DED` | משני |
| Orange | `#F08700` | מבטא / ראשי לארגונים |
| Rose | `#CA5369` | שגיאות, ביצועים נמוכים |

## מבנה הפרויקט
```
src/app/
  pages/          ← דפי הדשבורד
  components/     ← קומפוננטים משותפים
  data/
    mockData.ts   ← נתונים — החלף בנתונים אמיתיים
```

## הנחיות כלליות
- כל הממשק RTL עברית — `dir="rtl"` על האלמנט השורשי
- פונט: Rubik (Google Fonts)
- ספריות: React + Vite + TypeScript + Tailwind CSS v4 + shadcn/ui + Recharts
- ניווט: react-router v7
- פריסה: Vercel

## ענפים
- `main` — דשבורד Yeda LMS (דוגמה מלאה)
- `template` — תבנית גנרית להתחלה חדשה
